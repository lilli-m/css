CSS

Was ist CSS?

Cascading Style Sheets ist eine Gestaltungs- und Formatierungssprache, mit dieser Sprache kann man HTML Elemente stylen. CSS wird von allen Browser unterstützt und ist somit ein Web-Standart, den Standart definiert die World Wide Web Consortium. CSS wird verwendet, um das Design, die Farben, Schriftarten, Abstände, Größen und Positionierung von Elementen auf einer Webseite zu steuern.


In HTML einbinden und die Syntax eines CSS Befehls

Die grundlegende Syntax von CSS besteht aus einem Selektor und einer oder mehreren Eigenschaften, die auf das ausgewählte Element angewendet werden. Der Selektor wählt ein oder mehrere HTML-Elemente aus, auf die die Eigenschaften angewendet werden sollen.Cascading Style Sheets ist eine Gestaltungs- und Formatierungssprache, mit dieser Sprache kann man HTML Elemente stylen. CSS wird von allen Browser unterstützt und ist somit ein Web-Standart, den Standart definiert die World Wide Web Consortium. CSS wird verwendet, um das Design, die Farben, Schriftarten, Abstände, Größen und Positionierung von Elementen auf einer Webseite zu steuern.


Hier noch ein paar Beispiel-Befehle:

- color: legt die Farbe des Textes fest. Beispiel: color: red;

- font-size: legt die Größe des Textes fest. Beispiel: font-size: 16px;

- background-color: legt die Hintergrundfarbe eines Elements fest. Beispiel: background-color: #fff;

- margin: legt den Abstand zwischen dem Element und anderen Elementen fest. Beispiel: margin: 10px;

- padding: legt den Abstand zwischen dem Inhalt des Elements und dem Rand fest. Beispiel: padding: 5px;

- border: legt die Größe, Farbe und Art des Rahmens eines Elements fest. Beispiel: border: 1px solid black;

- text-align: legt die Ausrichtung des Textes innerhalb des Elements fest. Beispiel: text-align: center;

- display: legt die Art der Darstellung eines Elements fest. Beispiel: display: block;

- float: legt die Ausrichtung eines Elements innerhalb seines Elterncontainers fest. Beispiel: float: left;

- position: legt die Position eines Elements innerhalb seines Elterncontainers fest. Beispiel: position: absolute;
